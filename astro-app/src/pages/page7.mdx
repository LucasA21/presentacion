# ¿Qué es shadcn/ui?

- **Colección de componentes React** (Button, Dialog, Input, etc.)
- Basados en **Radix UI** y estilizados con **Tailwind CSS**.
- Se instalan mediante **CLI** y se **copian al proyecto**, quedando su código completamente editable.

```tsx
import { Button } from '@/components/ui/button';
<Button>Enviar</Button>
```
