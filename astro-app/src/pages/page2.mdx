# ¿Para qué sirve Tailwind y qué problemas resuelve?

- **Agiliza la maquetación**: no cambias entre HTML y CSS; aplicas utilidades in situ.
- **Evita CSS inflado**: el proceso de build elimina clases no usadas.
- **Consistencia**: utilidades basadas en un sistema de diseño uniforme.
- **Mantenibilidad**: no hay nombres de clase inventados ni conflictos de estilo.

```html
<div class="max-w-md mx-auto p-6 bg-white rounded-lg shadow">
  <h2 class="text-2xl font-semibold mb-4">Card</h2>
  <p>Contenido de ejemplo.</p>
</div>
```
