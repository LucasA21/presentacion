# Tailwind + shadcn/ui en acción

```tsx
import { Button, Input, Label } from '@/components/ui';

function LoginForm() {
  return (
    <form className="max-w-sm mx-auto p-6 bg-gray-100 rounded-lg shadow">
      <div className="mb-4">
        <Label htmlFor="email">Correo electrónico</Label>
        <Input type="email" id="email" placeholder="tu@correo.com" />
      </div>
      <Button type="submit" className="w-full">Ingresar</Button>
    </form>
  );
}
```

Integramos las utilidades de Tailwind para layout y espaciado con componentes accesibles de shadcn/ui.
