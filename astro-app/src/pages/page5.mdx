# Curva de aprendizaje

- **Inicial**: moderada — memorizar utilidades y acostumbrarse al HTML lleno de clases.
- **Productividad**: crece rápido; tras unas semanas ya dominas `p-4`, `text-lg`, `flex`, etc.
- **Documentación**: excelente, con ejemplos y herramientas de autocompletado.

> Mi experiencia: tras crear un par de componentes, escribir Tailwind es más rápido que el CSS clásico.
