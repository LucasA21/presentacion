**Modificar un componente existente**  
   - Como los componentes viven en el proyecto **(se instalan como archivo plano)**, se puede editar libremente, gracias a esto se le puede dar
   un estilo personalizado a cada componente.
   - **Ejemplo**: Cambiar el estilo del componente **Button** para usar **bg-button-primary**:

```ts
export function Button({ variant, size, ...props }: ButtonProps) {
  const baseStyles = 'inline-flex items-center justify-center rounded-md font-medium transition-colors focus:outline-none focus:ring-2 focus:ring-offset-2';
  const variantStyles: Record<string, string> = {
    primary: 'bg-blue-600 text-white hover:bg-blue-700 focus:ring-blue-500',
    primary: 'bg-button-primary text-white hover:bg-brand-secondary focus:ring-brand-primary',
    secondary: 'bg-gray-100 text-gray-900 hover:bg-gray-200 focus:ring-gray-300',
    destructive: 'bg-red-600 text-white hover:bg-red-700 focus:ring-red-500',
  };
}    
```

**Uso de variantes dinámicas**  
   - Shadcn UI permite variantes predefinidas **(Button: “primary”, “secondary”)**. Se puede añadir más variantes:
     ```ts
     // src/components/ui/button/component.tsx
     const variantStyles: Record<string, string> = {
       primary: 'bg-brand-primary text-white hover:bg-brand-secondary focus:ring-brand-primary',
       secondary: 'bg-gray-100 text-gray-900 hover:bg-gray-200 focus:ring-gray-300',
       info: 'bg-teal-500 text-white hover:bg-teal-600 focus:ring-teal-500', // nueva variante
     };
     ```
