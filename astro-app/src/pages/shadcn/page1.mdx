  <CardContent>
    <p class="">
      **Shadcn UI** es un conjunto de componentes accesibles y personalizables para React y Next.js, construido sobre **Radix UI** 
      y estilizado con **Tailwind CSS**.
    </p>
    <p class="mb-6">
      Se lanzó en marzo de 2023 y **combina** la filosofía de utilidades atómicas de **Tailwind** con la accesibilidad de **Radix** para ofrecer UI **“copy-and-paste”**
    </p>
    <p class="mt-6">
      Shadcn UI funciona como una **plataforma de distribución de componentes**: cada componente se crea en archivos planos y se instala en el proyecto mediante una CLI, lo que facilita la personalización o reemplazo de cualquier parte del código fuente​
    </p>
