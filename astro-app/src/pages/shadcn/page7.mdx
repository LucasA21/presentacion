<div class="max-w-4xl mx-auto p-6 bg-gradient-to-r from-purple-800 via-indigo-700 to-blue-600 text-white rounded-lg shadow-lg">
  <div class="mb-6 text-2xl font-bold">Herramientas comparables a Shadcn UI</div>

  <div class="flex flex-col md:flex-row gap-6">
    <div class="flex-1 bg-white/20 p-4 rounded-lg">
      <h3 class="text-xl font-semibold mb-2">Material UI (MUI)</h3>
      <p class="italic mb-2">Componentes React basados en Material Design.</p>
      <ul class="mb-2 list-disc list-inside text-sm">
        <li><strong>Pros:</strong> maduro, amplio ecosistema de temas.</li>
        <li><strong>Contras:</strong> overrides de Emotion para personalizar; bundle grande.</li>
      </ul>
    </div>

    <div class="flex-1 bg-white/20 p-4 rounded-lg">
      <h3 class="text-xl font-semibold mb-2">Chakra UI</h3>
      <p class="italic mb-2">UI React con theming por props.</p>
      <ul class="mb-2 list-disc list-inside text-sm">
        <li><strong>Pros:</strong> accesibilidad integrada, API declarativa.</li>
        <li><strong>Contras:</strong> no usa utilidades Tailwind; bundle grande.</li>
      </ul>
    </div>

    <div class="flex-1 bg-white/20 p-4 rounded-lg">
      <h3 class="text-xl font-semibold mb-2">Headless UI</h3>
      <p class="italic mb-2">Componentes sin estilos de Tailwind Labs.</p>
      <ul class="mb-2 list-disc list-inside text-sm">
        <li><strong>Pros:</strong> libertad total de diseño, muy ligero.</li>
        <li><strong>Contras:</strong> debes construir todo el CSS tú mismo.</li>
      </ul>
    </div>

    <div class="flex-1 bg-white/20 p-4 rounded-lg">
      <h3 class="text-xl font-semibold mb-2">DaisyUI / Flowbite</h3>
      <p class="italic mb-2">Plugins Tailwind con componentes preestilizados.</p>
      <ul class="mb-2 list-disc list-inside text-sm">
        <li><strong>Pros:</strong> instalación sencilla, temas listos.</li>
        <li><strong>Contras:</strong> menos flexibilidad que Shadcn UI, dependes de clases predefinidas.</li>
      </ul>
    </div>
  </div>

  <div class="mt-8 p-4 bg-white/10 rounded-lg">
    <p class="mb-2"><strong>¿Cuál gana en la comunidad?</strong></p>
    <ul class="list-disc list-inside text-sm">
      <li><strong>Shadcn UI</strong> (88 000 ⭐) supera a Material UI (85 000 ⭐) y Headless UI ( 27 000 ⭐).</li>
      <li><strong>En descargas npm</strong> , Material UI (1 400 000/sem.) y Chakra UI (1 000 000/sem.) están arriba, pero Shadcn UI crece rápido por su modelo de “copy-paste” de componentes.</li>
      <li><strong>Shadcn UI lidera por ⭐</strong>  en repositorios modernos de accesibilidad + Tailwind, mientras Material UI y Chakra siguen fuertes en proyectos sin Tailwind.</li>
    </ul>
  </div>
</div>
