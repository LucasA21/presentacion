# ¿Qué es Tailwind CSS?

Tailwind CSS es un **framework de CSS** con un enfoque *utility‑first*. Proporciona clases atómicas como:

- `bg-blue-500` (fondo azul)
- `text-white` (texto blanco)
- `mt-4` (margen superior)

En lugar de componentes prefabricados, combinas estas utilidades directamente en tu HTML:

```html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded">
  Click me
</button>
```

Esto te permite construir diseños personalizados sin escribir CSS manualmente.
