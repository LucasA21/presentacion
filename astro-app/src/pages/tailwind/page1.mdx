# ¿Qué es Tailwind CSS?

- Framework **utility-first** para CSS.
- Proporciona clases atómicas (p. ej. `flex`, `mt-4`, `text-center`).
- Permite construir diseños personalizados **directamente en el markup**.

```astro
<div class="p-4 bg-primary text-white">
  <h1 class="text-3xl font-bold">Hola Mundo</h1>
</div>
```
<div class="p-4 bg-gradient-to-r from-purple-800 via-indigo-700 to-blue-600 text-white">
  <h1 class="text-3xl font-bold">Hola Mundo</h1>
</div>
