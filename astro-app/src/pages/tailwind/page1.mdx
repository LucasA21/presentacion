# ¿Qué es Tailwind CSS?

- Framework **utility-first** para CSS, creado en 2017 como alternativa a frameworks basados en componentes predefinidos (Bootstrap, Bulma, etc.).
- Proporciona clases atómicas (p. ej. `flex`, `mt-4`, `text-center`) para escribir estilos directamente en el elemento html.
- Su compilador **JIT** (Just-In-Time) genera dinámicamente únicamente las clases usadas, resultando en un CSS final muy ligero.

```astro
<div class="p-4 bg-primary text-white">
  <h1 class="text-3xl font-bold">Hola Mundo</h1>
</div>
```
<div class="p-4 bg-gradient-to-r from-purple-800 via-indigo-700 to-blue-600 text-white">
  <h1 class="text-3xl font-bold">Hola Mundo</h1>
</div>
