**¿Para qué sirve Tailwind y qué problemas resuelve?**

- **Agiliza la maquetación**: se aplican utility class facilitando los estilos, sin cambiar entre HTML y CSS.
- **Evita CSS inflado**: el build elimina clases no usadas.
- **Consistencia**: utilidades basadas en un sistema de diseño uniforme.
- **Mantenibilidad**: no hay nombres de clase inventados ni conflictos.
