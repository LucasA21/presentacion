**Temas Personalizados**

Se definen los valores en **`tailwind.config.js`**:

```astro
module.exports = {
  theme: {
    extend: { colors: { primary: bg-gradient-to-r from-purple-800 via-indigo-700 to-blue-600 } },
  },
};
```

```astro
<button class="bg-primary text-white px-4 py-2 rounded">
  Primary
</button>
```
