# Comparativa con otras herramientas

**CSS puro**
- Máximo control, pero laborioso y propenso a duplicar reglas.

**Bootstrap**
- Componentes listos, fácil de arrancar, pero diseño rígido y CSS voluminoso.

**Bulma / Foundation**
- Opciones modernas pero con componentes predefinidos y menor flexibilidad.

**Librerías React (MUI, Chakra)**
- Componentes completos con JavaScript, menos centrados en personalización de CSS.

**Utility‑first** (Tachyons, UnoCSS)
- Similares a Tailwind, pero con ecosistema y adopción menores.

> Tailwind destaca por su balance entre flexibilidad, rendimiento y ecosistema.
