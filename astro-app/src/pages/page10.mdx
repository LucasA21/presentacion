# shadcn/ui vs Material UI y Chakra UI

**Material UI**
- Paquete npm, estilo Material, componentes cerrados.

**Chakra UI**
- Style‑props, theming integrado, componentes precompilados.

**shadcn/ui**
- Código abierto, editable, utiliza Tailwind para estilado.
